Making an element interactive with a tabindex? Name it!

Don't name an element nested in it. This does not work. The name needs to be on the element that you can -focus-.

Hiding a house number in the back of my garden does not help identify my house from the front.

https://www.w3.org/WAI/WCAG22/Understanding/name-role-value.html
