It's often the first thing I test on any website.

No technical skills or tools required. When a website opens press the tab-button.

A skip-link should appear.
(an interactive item stating something like "skip to content")

If it doesn't, that's probably a failure of WCAG SC 2.4.1 Bypass Blocks.
